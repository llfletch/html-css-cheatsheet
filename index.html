Cheat Sheet for Tables in HTML

Elements Stands For Description
<table> The highest-level element of a table. It encloses all the other elements.
<tr> table row Each row of a table is surrounded by this element (including the headings).
<th> table heading Each heading (even empty ones) must be enclosed by this tag.
<td> table data It is used to represent an individual cell. It encloses the table data.
<thead> table head It should enclose the tags for the headings of long tables.
<tbody> table body It should enclose the tags for the body of long tables.
<tfoot> table footer It should enclose the tags for the footer of long tables.
rowspan The rowspan attribute is used when we want one cell in the table to span multiple rows.
colspan The colspan attribute is used when we want one cell in the table to span multiple columns.
